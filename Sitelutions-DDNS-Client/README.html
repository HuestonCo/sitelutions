Sitelutions Dynamic DNS (DDNS) Client for Windows

This is the official Sitelutions DDNS client for Windows. It is a lightweight application that runs in your system tray, automatically detects your public IP address, and updates your Sitelutions DNS records.

Features

  • Automatic Updates: Set and forget. Updates your IP at regular intervals (60 minutes, 4 hours, 6 hours, or 24 hours).
  • System Tray Icon: Hides to the system tray (notification area) to stay out of your way.
  • Saves Settings: Remembers your Record ID, Email, and API Key securely in your user's AppData folder.
  • Activity Logging: A simple log window shows you the history of update attempts and server responses.
  • Manual Update: A "Update Now" button to force an IP check and update immediately.
  • Lightweight: Built with Python, it has minimal system resource usage.

How to Find Your Record ID

The "Record ID" is required for the application to know which DNS record to update. You can find this ID in your Sitelutions dashboard:

  1. Log in to your Sitelutions account.
  2. Navigate to DNS Management from the side menu.
  3. Find the domain you wish to update (e.g., yourdomain.com) and click the three-dot menu (...) on the right.
  4. Select Manage from the dropdown.
  5. On the "DNS Records" page, locate the specific A Record you want to keep updated (e.g., home.yourdomain.com).
  6. Click the three-dot menu (...) on the far right of that record's row.
  7. Select Dynamic DNS from the dropdown.
  8. The next page will display the "Record identifier". This numeric ID is what you will paste into the "Record ID" field of the DDNS client.

Installation & Usage

You have two options to use this application: downloading the pre-compiled .exe (recommended for most users) or running the script from the source code (for developers).

Option 1: For End-Users (Recommended)

  1. Go to the Releases page of this repository.
  2. Download the latest Sitelutions-DDNS-Updater.exe file.
  3. Place the .exe file in any folder on your computer and run it. The application will start and your settings will be saved automatically.

Option 2: For Developers (Running from Source)

If you have Python installed, you can run the application directly from the source code.

1. Install Dependencies:
You will need Python 3 and a few libraries. You can install them using pip:

pip install requests pystray pillow

2. Run the Application:
Clone this repository (or download the files) and run the following command from the script's directory:

python ddns_updater.py

(Note: You may need to use python3 instead of python depending on your system's setup.)

For Developers & Contributors

How to Build the .exe Yourself

If you want to modify the code and compile your own .exe file, you will need PyInstaller.

1. Install PyInstaller:

pip install pyinstaller

2. Run the Build Command:
From the project's root directory (where ddns_updater.py, icon.ico, and icon.png are located), run the following command:

pyinstaller --onefile --noconsole --name SitelutionsUpdater --icon="icon.ico" --add-data="icon.png;." ddns_updater.py
  • --onefile: Bundles everything into a single executable.
  • --noconsole: Prevents the black command-line window from opening (essential for tray apps).
  • --name: Sets the output filename.
  • --icon="icon.ico": Sets the application and .exe file icon.
  • --add-data="icon.png;.": Crucial Step. This bundles the icon.png file, which is required by the system tray icon.

Your new .exe file will be located in the dist folder.

Direct API (GET) Usage for Custom Scripts

This application uses the api2.sitelutions.com endpoint. You can use this same endpoint for your own custom scripts, routers, or other devices. The API is designed for simplicity and compatibility, using a standard HTTP GET request.

Endpoint URL Format:

https://api2.sitelutions.com/dnsup?user=[YOUR_EMAIL]&pass=[YOUR_API_KEY]&id=[YOUR_RECORD_ID]&ip=[NEW_IP_ADDRESS]&ttl=60

Parameters:

  • user: Your Sitelutions login email.
  • pass: Your API Key (generated in your Sitelutions account panel).
  • id: The specific DNS Record ID(s) you want to update.
  • ip: The new IP address you want to set.
  • ttl: The Time-To-Live in seconds (e.g., 60).

Example (using cURL in a bash script):

# 1. Get your current public IP from the Sitelutions IP service
MY_IP=$(curl -s https://api2.sitelutions.com/myip)

# 2. Set your variables
EMAIL="your_email@your_domain.com"
API_KEY="YOUR_API_KEY"
RECORD_ID="99234"

# 3. Send the update request
RESPONSE=$(curl -s "https://api2.sitelutions.com/dnsup?user=${EMAIL}&pass=${API_KEY}&id=${RECORD_ID}&ip=${MY_IP}&ttl=60")

echo "Server Response: $RESPONSE"
